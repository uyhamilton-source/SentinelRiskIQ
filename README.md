
# RiskNavigator Combined Compliance App

GitHub + Streamlit-ready app that auto-detects SOC 2 vs HIPAA rows, supports a toggle view (SOC 2 / HIPAA / Combined), generates a combined executive summary, and exports a client-ready PDF.

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Cloud
- Upload all files in this folder to a GitHub repo
- Deploy `app.py` on Streamlit Cloud

Demo login: `admin / admin123`
